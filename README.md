# Spacey

A React + Vite template for creating a spacey frontend developer portfolio.

Chiptune music made with [Tone.js](https://tonejs.org/).

## Setup

    npm install
    npm run dev

## Build

    npm run build

## Preview

    npm run preview

## Deploy

    npm run build
    npm run build:docs
    npm run preview

## Deploy to Github Pages

    npm run build:docs  # Build for Github Pages
